# SkillPath AI

SkillPath AI is an AI-powered career recommendation platform that helps users identify suitable career paths based on their skills and interests. The application provides personalized career suggestions, learning roadmaps, interview preparation resources, and job recommendations through a simple and interactive web interface.

---

## Features

- Career recommendations based on user skills
- NLP-based skill analysis
- Personalized learning roadmap
- Interview preparation section
- Job recommendations
- Responsive and user-friendly interface

## How It Works

1. The user enters their skills and interests.
2. The system processes the input using NLP techniques.
3. Relevant career paths are identified.
4. A personalized learning roadmap is generated.
5. Interview preparation resources and job suggestions are displayed.

## Installation

Clone the repository

```bash
git clone https://github.com/kanmani70/skillpath-ai.git
```

Go to the project directory

```bash
cd skillpath-ai
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```
